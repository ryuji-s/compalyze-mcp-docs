Compalyze MCP — 日本企業データを AI から使う

Compalyze は、法人番号ベースで日本企業を広く収録し、登記・財務・企業間の関係・役員などのデータを、公式情報源（登記情報・EDINET・官報など）から構造化して提供する企業情報プラットフォームです。 この MCP サーバーを使うと、Claude などの AI エージェントから直接これらの企業データを取得できます。

compalyze.co.jp を見る MCP ダッシュボード / APIキー発行

これは何か

Compalyze MCP は Model Context Protocol 準拠のリモート MCP サーバーです。 「企業を探す（Discovery）」と「個社を深く知る（Intelligence）」の読み取りツール群を提供し、営業リスト作成・与信/取引先調査・競合や業界の分析・AI アシスタントからの企業データ取得などに使えます。 Web で人が使う場合は compalyze.co.jp（会社ページ・詳細検索）をどうぞ。

クイックスタート（接続）

リモート HTTP トランスポートで接続します。cpz_ で始まる API キーを Authorization: Bearer ヘッダで渡します。

Claude Code

claude mcp add compalyze https://mcp.compalyze.co.jp/mcp \
  --transport http \
  --header "Authorization: Bearer cpz_xxxxxxxxxxxxxxxx"

Claude Desktop（claude_desktop_config.json）

{
  "mcpServers": {
    "compalyze": {
      "type": "http",
      "url": "https://mcp.compalyze.co.jp/mcp",
      "headers": { "Authorization": "Bearer cpz_xxxxxxxxxxxxxxxx" }
    }
  }
}
API キーは MCP ダッシュボード で発行します。キーは cpz_ プレフィックス付きで、サーバー側にはハッシュのみ保存されます（平文は発行時のみ表示）。

認証と料金

  • 認証：API キー（cpz_）。HTTP モードは Authorization ヘッダ、stdio モードは COMPALYZE_API_KEY 環境変数。
  • 料金：固定月額 ＋ クレジット従量制。ツール呼び出しごとにクレジットを消費します。プラン・残量は ダッシュボードで確認できます。

AI エージェント向けの従量決済（x402）

AI エージェント向けに、ウォレット決済（x402 / USDC on Base）による従量アクセスにも対応しています（エージェント市場経由）。決済は facilitator を介して処理されます。

データソース

登記情報・EDINET・官報（決算公告）などの公式情報源をもとに、日本企業を広く収録して構造化しています（収録範囲はデータ種別により異なります）。 個社の詳細は compalyze.co.jp の会社ページで確認できます。
